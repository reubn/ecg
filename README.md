# ecg

-----
ESP8266-based WiFi connected ECG device. Portable.

Right Arm = <span style="color:red">Red</span> = -ve

Left Arm = <span style="color:green">Green</span> = +ve

Right Leg = <span style="color:yellow">Yellow</span> = leg drive
